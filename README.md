# Spotify-style Deezer + Deemix desktop client

An Electron desktop music client that recreates the feel and core UX of the Spotify desktop app, but uses Deezer as the backend and Deemix-style downloads for ad-free playback and offline files.

Not affiliated with Spotify or Deezer.

Docs (screenshot, architecture, current status): [docs/README.md](docs/README.md)

## Run

- Desktop app (Electron):
  - `npm install`
  - `npm start`
